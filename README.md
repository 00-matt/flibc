# flibc

Freestanding libc implementation.
Useful for operating system kernels, bootloaders, small statically
linked applications, or as a starting point for your own libc.

## Implemented Methods

### `ctype.h`

* `isspace`

### `stdlib.h`

* `itoa`

### `string.h`

* `memcmp`
* `memcpy`
* `memmove`
* `memset`
* `strlen`
* `strnlen`
